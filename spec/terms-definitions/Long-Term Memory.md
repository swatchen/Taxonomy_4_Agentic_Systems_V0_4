[[def: Long-Term Memory]]

~ ~Persistent historical and contextual information stored at the Agent Role level (e.g., Workload Execution Records, curated knowledge, preferences, policies), explicitly decoupled from any single Live Agent run.

~ ### **Low-Embeddedness Component—** A component that “behaves similarly across many contexts,” with behavior/risk largely determined by its internal logic (e.g., deterministic parser; validation subroutine; model wrapped in a narrow pattern), making it a natural candidate for global certification and reuse.
